## Write a Java program to create an array of its anti-diagonals from a given square matrix

Example:
Input :

[

[1 2]

[3 4]

]

Output:

[

[1],

[2, 3],

[4]

]

Input:

[10, 20, 30]

[50, 60, 70]

[90, 100, 110]

Output:

[10]

[20, 50]

[30, 60, 90]

[70, 100]

[110]
