## Find all triplets equal to a given sum in a unsorted array of integers

Write a Java program to find all triplets equal to a given sum in a unsorted array of integers.

Example:

Input :

nums = { 1, 6, 3, 0, 8, 4, 1, 7 }

Output:

Triplets of sum 7

(0 1 6)

(0 3 4)
