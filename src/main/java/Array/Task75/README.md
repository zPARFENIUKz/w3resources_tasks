##  Largest gap between sorted elements of an array
Write a Java program to calculate the largest gap between sorted elements of an array of integers.

Sample Data:
{1, 2 ,9, 0, 4, 6} -> 3
{23, -2, 45, 38, 12, 4, 6} -> 15
